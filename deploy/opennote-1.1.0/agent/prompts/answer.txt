请从当前页面中提取所有正式答案和解析，不要提取页眉、页脚、目录和无关说明。

只输出 JSON 数组，不要输出 Markdown。

字段：
- no: 答案解析前方独立显示的题号，只保留数字字符串，例如 "12"；不要把年份、省份、页码当成题号。如果没有独立题号，请按本页答案解析从上到下填写 "1"、"2"、"3"。
- answerLetter: 答案字母，取值 A / B / C / D；无法判断则为空字符串。
- text: 答案字母与完整解析全文。
- type: 题型，取值 figure / definition / analogy / logic / other。
- sectionTitle: 当前章节或小节标题，无法判断则为空字符串。
- bbox: 当前答案解析在整页图片中的大致位置，使用归一化坐标 [x1,y1,x2,y2]，范围 0-1；无法判断则为 null。

输出示例：
[
  {
    "no": "1",
    "answerLetter": "A",
    "text": "A。【解析】元素组成相同，优先考虑位置规律……",
    "type": "figure",
    "sectionTitle": "图形推理",
    "bbox": [0.07, 0.18, 0.93, 0.38]
  }
]

如果当前页没有正式答案解析，返回 []。
